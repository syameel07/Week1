Day 1: Introduction to Web Development (17 December 2024)

Starting my web development journey was exciting! Today, I jumped into the basics of HTML and CSS, setting up the groundwork for web pages. Mastering Linux commands felt like learning to talk to my computer and I uncovered the role of HTML metadata and basic tags in structuring content.

Highlights:

  • -- Mastered basic LINUX commands to navigate and manage files.
  • -- Understood HTML meta data, crucial for providing information about the web page.

Day 2: Diving Deeper into HTML & CSS (19 December 2024)

Building on the basics, today was all about adding more depth to my web pages. I learned about HTML attributes that add extra functionality to our tags and basic styling techniques using CSS. Navigating web pages became more intuitive as I understood how to create links between different sections and pages.

Highlights:

  • -- Learned about HTML attributes to enhance the functionality of tags.
  • -- Started with basic styling, making web pages visually appealing.
  • -- Created navigation menus for seamless page transitions.
